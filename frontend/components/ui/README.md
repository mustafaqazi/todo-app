# shadcn/ui Components

This directory contains shadcn/ui components that need to be installed via:

```bash
npx shadcn-ui@latest init
npx shadcn-ui@latest add button card input table dialog dropdown-menu toast checkbox badge separator avatar sheet tooltip skeleton
```
